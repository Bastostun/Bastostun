### 🌌 Featured Project: CloudSmartTutor (云端智师)

> **"Breaking public cloud monopoly by redefining legacy hardware as standard Edge Micro-IDC compute nodes."**

`CloudSmartTutor` is a Distributed Edge Hyper-Converged Micro-IDC platform. It injects white-box **AI Agents** directly into Proxmox VE sandboxes via non-intrusive Go daemons. Instead of copying error logs, the ambient AI automatically diagnoses compile errors and routing anomalies from raw system `stderr`, generating actionable execution cards while optimizing memory consumption up to 60% through cluster KSM hooks.

- **Frontend Core:** Built with Vue 3, Vite, TypeScript, and fluid glassmorphism dashboards.
- **Backend Hub:** Powered by Go (Gin Framework) with thread-safe connection pools and in-memory clustering algorithms.
